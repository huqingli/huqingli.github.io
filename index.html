PyQt5简介与安装

PyQt5简介

PyQt5是Digia公司开发的Qt5应用的Python版本，同时支持python 2.x和python 3.x。本教程采用python 3.7。Qt库是最强大的GUI库之一。PyQt5的出现，使得开发者能够利用python去调用Qt库，大大提高了开发效率。PyQt5包含超过620个类，6000个函数和方法。PyQt5具有跨平台特性，能够在Unix，Windows和Mac OS上运行。

注：PyQt5是双重许可的，开发人员可以在GPL和商业许可之间进行选择。

PyQt5模块

PyQt5的类被分为如下几个模块：

  • QtCore
  • QtGui
  • QtWidgets
  • QtMultimedia
  • QtBluetooth
  • QtNetwork
  • QtPositioning
  • Enginio
  • QtWebSockets
  • QtWebKit
  • QtWebKitWidgets
  • QtXml
  • QtSvg
  • QtSql
  • QtTest

QtCore包含一些与GUI无关的核心功能，这个模块常用于处理时间、文件和目录、各种数据类型、流、URL、MIME类型、线程或进程。

QtGui包含用于窗口系统集成、事件处理、2D图形、基本成像以及字体和文本的类。

QtWidgets包含一组用于创建桌面用户界面的UI元素的类。

QtMultimedia包含用于处理多媒体内容的类和用于访问摄像机和无线电功能的类。

QtBluetooth模块包含扫描设备并与之连接和交互的类。

QtNetwork模块包含用于网络编程的类。这些类通过使网络编程更容易和更便携来促进TCP/IP和UDP客户端和服务器的编码。

QtPositioning包含通过使用各种可能的来源（包括卫星，Wi-Fi或文本文件）来确定位置的类。

Enginio模块实现了用于访问Qt云服务托管应用程序运行时的客户端库。

QtWebSockets模块包含实现WebSocket协议的类。

QtWebKit包含基于WebKit2库的Web浏览器实现的类。

QtWebKitWidgets包含基于WebKit1的Web浏览器实现的类，用于基于QtWidgets的应用程序。

QtXml包含用于处理XML文件的类。该模块为SAX和DOM API提供了实现。

QtSvg模块提供用于显示SVG文件内容的类。可缩放矢量图形（SVG）是一种用于描述XML中的二维图形和图形应用程序的语言。

QtSql模块提供了用于处理数据库的类。

QtTest包含可以对PyQt5应用程序进行单元测试的函数。

PyQt5与PyQt4的区别

PyQt5不向后兼容PyQt4，有几个重大变化。但是，将旧代码调整到新库并不是很困难。差异如下：

  • Python模块已经重组。一些模块已被删除（QtScript），其他模块已被拆分为子模块（QtGui，QtWebKit）。
  • 新模块已经推出，包括QtBluetooth，QtPositioning或Enginio.
  • PyQt5仅支持新式信号和插槽处理。不再支持对SIGNAL（）或SLOT（）函数的调用.
  • PyQt5不支持在Qt v5.0中标记为已弃用或已过时的Qt API的任何部分。

PyQt5安装

本教程使用python3，通过pip3进行安装PyQt5，因此在安装之前必须先安装python，这里推荐使用anaconda安装和管理python。Anaconda官网：https://www.anaconda.com/distribution/

pip3 install pyqt5
pip3 install pyqt5-tools

PyQt5最新版本将两者分开，qtdesigner.exe文件放在pyqt5-tools文件夹中，因此在使用pycharm进行开发时，需要配置qt tool的位置。
